# FrameSplitter

nフレームごとにレイヤーを分割。

## 使い方

![UI](ui.png)

1. **Frame**で何フレームごとに分割するかを決める。

1. **Split**をクリックで実行。**Remove**がチェックされている場合、元の動画は取り除かれる。
